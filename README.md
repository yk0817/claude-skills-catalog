# Claude Code Skills Catalog

インストール済みClaude Codeスキルの一覧とパーソナライズ推薦を表示するGitHub Pagesサイト。

## Features

- 全30スキルをカード形式で表示（10カテゴリ）
- カテゴリフィルター・関連度ソート
- ユーザープロフィールに基づくパーソナライズ推薦 TOP 5
- ダークテーマ・レスポンシブデザイン

## Deploy

GitHub Pages設定で `docs/` フォルダを指定してください。

## Structure

```
docs/
└── index.html    ← 単一HTML（CSS/JS埋め込み）
```
